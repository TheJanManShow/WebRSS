AI Summaries — Local LLM

Connect to a locally running Ollama instance or any OpenAI-compatible API (LM Studio, Jan, etc.) to generate private article summaries. Your article text is sent only to your local endpoint — nothing goes to external servers.

⚠ Setup required — read this if the connection fails
Browsers block a secure https:// page (like GitHub Pages) from calling an insecure http://localhost endpoint — this is called mixed-content blocking. Two ways around it:
  1. Run WebRSS locally over http://localhost (e.g. python -m http.server) — then both sides are http and it works directly.
  2. Or put a small HTTPS reverse-proxy in front of Ollama and use that https:// URL here.
Also: Ollama must allow this site as an origin. Start it with
OLLAMA_ORIGINS=* ollama serve
(or set it to this page's exact origin). Click Test connection below to verify.
Ollama default: http://localhost:11434/v1/chat/completions
